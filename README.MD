# Pharmacy Management System
The Pharmacy Management System is a web-based application designed to manage daily pharmacy operations efficiently. It helps in handling medicine inventory, sales, suppliers, and customer records while reducing manual work and errors.

- **Dhiraj Dhar Dip** (ID: 232-134-019)

## Project Objectives
- Automate pharmacy inventory and billing process
- Track medicine stock and expiry dates
- Maintain accurate sales and purchase records
- Improve efficiency and data security

## Core Features
- Secure login with role-based access (Admin, Pharmacist)
- Medicine management (add, update, delete, search)
- Stock monitoring with low-stock and expiry alerts
- Sales billing and invoice generation
- Supplier and customer management
- Sales and inventory reports

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Server**: XAMPP / WAMP

## System Modules
- User Management
- Medicine & Inventory Module
- Sales & Billing Module
- Supplier & Customer Module
- Report Module

## Installation Steps
1. Clone the repository
2. Import the database (.sql file) into MySQL
3. Configure database connection
4. Run the project on localhost using XAMPP

## Conclusion
This system provides a reliable and user-friendly solution for managing pharmacy operations and is suitable for academic and small-scale real-world use.
